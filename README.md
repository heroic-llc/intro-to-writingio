# Intro to writingIO

An onboarding video series introducing new users to writingIO.

## Course Overview

This short video series welcomes new users to writingIO and helps them understand what the platform offers, how learning works, the tools available, and how to get started.

## Course Structure

```
intro-to-writingio/
├── README.md
├── CLAUDE.md
└── video-scripts/
    ├── lesson-1-what-is-writingio.md
    ├── lesson-2-learning-with-writingio.md
    ├── lesson-3-the-tools.md
    └── lesson-4-youre-ready.md
```

## Lessons

1. **What is writingIO** - Introduction to Amanda (CLO), mission, credibility (100k+ users, Microsoft, Shopify, NASA)
2. **Learning with writingIO** - Interactive courses, certifications, badges, continuing education
3. **The Tools** - AI models dashboard (GPT, Claude, Gemini, Grok), document editor
4. **You're Ready** - The team behind writingIO, next steps (free AI Essentials course or jump into tools)

## Target Audience

- New writingIO users during onboarding
- Potential customers learning about the platform
- Anyone curious about what writingIO offers
